# AI-агенты Rocketmind — Маскоты

Персонажи-маскоты — часть дизайн-системы Rocketmind. Каждый агент представлен 3D-аватаром с характером, ролью и задачами. Используются на страницах сайта, в каталоге агентов, в аватарках чат-интерфейса и в блоках с призывом к взаимодействию.

## Правила использования в интерфейсе

- **Аватар в чате** — круглый контейнер `40–48px`, 3D-голова агента на фоне `--color-violet-10` или нейтральном
- **Карточка каталога** — аватар `80px` + имя + роль + короткое описание
- **Hero/CTA блок** — крупный аватар `120–160px` «выглядывает» из-за края блока или карточки (обрезается снизу/сбоку), создавая живость
- **Стикер/подпись** — маленький аватар `32px` рядом с Shantell Sans подписью
- Персонажи не симметричны и не статичны — всегда с жестом или наклоном
